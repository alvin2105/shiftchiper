## Shift Cipher atau disebut juga Caesar Cipher merupakan teknik enkripsi berbasis substitusi


- Enkripsi => E(x) = (x+K) mod 26
- Dekripsi => D(x) = (x-K) mod 26


Ket:

x = alfabet dalam angka (A-Z = 0-25)

K = kunci
